# FindFirstDuplicate Problem
Problem: Given an array of integers, find the first duplicate value.
If no duplicate exists, return -1.
---

## Examples

### Example 1
[2, 1, 3, 5, 3, 2] → Output: 3 (first duplicate)

### Example 2
Example: [1, 2, 3, 4] → Output: -1

#Test

print(first_duplicate([2, 1, 3, 5, 3, 2]))  # 3
print(first_duplicate([1, 2, 3, 4]))        # -1
